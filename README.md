# Glass-Bridge-Survival-Game-Project
The Glass Bridge Survival Game challenges players to cross a virtual bridge by memorizing and selecting safe glass panels. The game displays a safe path briefly, then players must recall and navigate the pattern correctly to advance.

OBJECTIVES:
• Implement a memory-based survival game using Finite State Machine (FSM) on
FPGA
• Use Linear Feedback Shift Register (LFSR) for random safe path generation
• Validate player choices using comparator logic with switch inputs
• Display game progress using LEDs and 7-segment displays
• Demonstrate practical digital logic design concepts on hardware
